GeneLookup.js

Genelookup.js is a JQuery plugin for searching and building lists of genes. Use this plugin to populate HTML text area inputs with lists of gene identifiers. GeneLookup.js is powered by the Mygene.info gene annotation web service.

Textarea Element:
Code:
$("#mytextarea").lookup();
